PROGRAM OVERVIEW

Zero-knowledge proving with Vault-mediated signature delegation

Production ZK proofs against secrets managed by HashiCorp Vault, without ever releasing the root key

The paradox this solves
ZK circuits need
The plaintext witness
⇄
Vault exists to
Never release plaintext
Resolution
Vault signs a short-lived, scope-bound delegation. The circuit proves knowledge of the delegation, not the root key.

Architecture at a glance

TRUSTED
Vault cluster
Holds sk_root, signs delegations, audits every issuance
UNTRUSTED
App + prover
Holds short-lived delegation, generates proof from it
PUBLIC
Verifier
Checks proof against pk_root and public inputs
Root key never exits Vault. Delegation is scope-bound and expires in minutes. Compromise of the prover leaks only the delegation, not the key.
